# TASK
Task: Time/Task Management Service
태스크: 일정/시간 관리 서비스

# Features
+ Daily Do 서비스
    + 오늘 할 일 관리 기능
    + 설정 - 요일별 할 일 추가 기능 (자동으로 해당 요일에 할 일이 추가됨)
    + 다크 모드
+ Pomodoro 서비스
    + 타이머 기능
    + 오늘 달성한 포모
+ Dashboard 서비스

# Tech Stacks
+ 서버
    + REST API
    + Java
    + Spring Boot
+ 웹 클라이언트
    + HTML5/CSS3
    + JavaScript (ES6)
    + React.js (Hooks)
    + Webpack, Babel
    + React Router
